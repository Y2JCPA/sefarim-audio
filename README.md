# 📚🎧 Sefarim Audio

Audiobook-style recordings of classic Jewish hashkafa texts, generated from [Sefaria](https://www.sefaria.org)'s open API using modern text-to-speech.

## How It Works

1. English text pulled from Sefaria's API
2. Cleaned and formatted for natural narration
3. Converted to MP3 using OpenAI TTS
4. One file per chapter — listen anywhere
